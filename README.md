# gulp-starter

## Install gulp globally:
```sh
$ npm install --global gulp
```

To use npm install ```sh --globall``` without sudo on osx : http://www.johnpapa.net/how-to-use-npm-global-without-sudo-on-osx/.

## Run gulp:
```sh
$ gulp
```

## Run gulp task:
```sh
$ gulp <task>
```

# Observe:
```sh
$ gulp watch
```
